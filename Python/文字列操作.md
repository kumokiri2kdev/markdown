# 文字列操作

## 文字列への配列インデックス
### N文字目の文字

``` python
s = "abcde"
In []: print(s[2])
c

In []: print(s[-2])
d
```
文字列長より大きい（もしくは、0より小さい）index を指定すると index error になる。

## ループ処理
### enumerate

``` python
: for i, c in enumerate(s):
    ...:     print("{} : {}".format(i, c))
    
0 : a
1 : b
2 : c
3 : d
4 : e
```

## 空白削除

``` python
In []: s = " x "

In []: print('A' + s.strip() + 'B')
AxB

In []: print('A' + s.lstrip() + 'B')
Ax B

In []: print('A' + s.rstrip() + 'B')
A xB

```
