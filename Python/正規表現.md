# 正規表現
## re
Python の正規表現では、re パッケージを使用する。

## 操作
### search

``` python
import re

target = "return doAction('/JRADB/accessO.html','pw15orl10042017020720170819/C3')"
pattern = r'\(.*\)'

searched = re.search(pattern, target)

print(searched[0])

```
