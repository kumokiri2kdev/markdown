一般的に、Zスコアが ± 3 より大きい場合に外れ値として取り扱う。

外れ値を取り除くには、下記の様にする。

``` Python
zscore = (ser - ser.mean())/ser.std(ddof=0)
ser = ser[(zscore >= -3) & (zscore <= 3)]
```
